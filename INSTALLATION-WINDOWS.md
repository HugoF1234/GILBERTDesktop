# Installation Gilbert Desktop sur Windows

## Étape 1 : Télécharger

1. Aller sur [GitHub Releases](https://github.com/HugoF1234/GILBERTDesktop/releases)
2. Télécharger le fichier `.msi` (ex. `Gilbert_1.3.1_x64_en-US.msi`)

## Étape 2 : Installer

1. Double-cliquer sur le fichier `.msi`
2. Si l’assistant d’installation s’ouvre directement → suivre les étapes
3. Si **SmartScreen** s’affiche → voir ci-dessous

## Si Windows SmartScreen s’affiche

Windows peut afficher une fenêtre **« Windows a protégé votre PC »** ou **« L’application a été bloquée pour votre protection »**.

**Pourquoi ?** L’app est signée, mais sans certificat EV (Extended Validation), Windows SmartScreen peut afficher cet avertissement pour les logiciels peu téléchargés.

**Comment continuer :**

1. Cliquer sur **« Informations complémentaires »** (ou « More info »)
2. Cliquer sur **« Exécuter quand même »** (ou « Run anyway »)
3. L’assistant d’installation se lance normalement

> L’app Gilbert est publiée officiellement et ne contient pas de logiciel malveillant. Cet avertissement est une mesure de sécurité standard de Windows pour les logiciels non encore largement reconnus.

## Après l’installation

- Gilbert est disponible dans le menu Démarrer et sur le bureau (si choisi lors de l’installation)
- Se connecter avec votre compte Gilbert
- **Note :** Sur Windows, seul le micro est disponible pour l’enregistrement (pas de capture du son système des apps de visio)
